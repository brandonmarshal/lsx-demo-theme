## Example Usage

### Creating a Hero Section

```
1. Insert > Patterns > "Section Style - Hero"
2. Replace the heading text with your own
3. Update the button links
4. Add a background image (optional)
5. Publish!
```

### Creating an Archive Grid

```
1. Insert > Patterns > "Section Style - Archive Grid"
2. Replace placeholder content
3. Add images to each column
4. Update the links
5. Publish!
```

## Customization

All styles can be customized using the WordPress block editor:

- **Change colors**: Use the color picker in block settings
- **Adjust spacing**: Modify padding and margins
- **Update typography**: Change font sizes and styles
- **Set alignment**: Choose content width (default, wide, full)

## Responsive Design

All section styles automatically adapt to different screen sizes:

- **Desktop**: Full padding and spacing
- **Tablet**: Optimized for touch
- **Mobile**: Reduced padding for better readability
